# MODUL 5 FLOW CONTROL: DECISION MAKING
**Tujuan Pembelajaran :**
1.	Mampu menjelaskan dan mengimplementasikan Flow Control: Pengambilan Keputusan dalam pemrograman menggunakan IDE.

**Penjelasan Singkat Materi:<br>**
**A. Pernyataan If<br>**
‚Ä¢	Definisi: Pernyataan if digunakan untuk mengeksekusi blok kode tertentu jika kondisi yang diberikan terpenuhi (benar). Jika kondisi tersebut tidak terpenuhi, program akan melanjutkan eksekusi ke baris berikutnya tanpa mengeksekusi blok tersebut.<br>
‚Ä¢	Sintaks: Sintaks untuk pernyataan if terdiri dari kata kunci if diikuti dengan kondisi dalam tanda kurung, dan blok kode yang akan dieksekusi jika kondisi tersebut benar diletakkan dalam kurung kurawal.
**B. Pernyataan If Else<br>**
‚Ä¢	Definisi: Pernyataan if else memperluas fungsi dari pernyataan if dengan menambahkan blok alternatif yang akan dieksekusi jika kondisi dalam if tidak terpenuhi (salah).<br>
‚Ä¢	Sintaks: Sintaks ini mirip dengan pernyataan if, tetapi setelah blok kode untuk kondisi benar, terdapat kata kunci else yang diikuti oleh blok kode yang akan dieksekusi jika kondisi salah.<br>
**C. Pernyataan If Else If<br>**
‚Ä¢	Definisi: Pernyataan if else if memungkinkan untuk memeriksa beberapa kondisi secara berurutan. Jika salah satu kondisi benar, blok kode yang sesuai akan dieksekusi.<br>
‚Ä¢	Sintaks: Dalam sintaks ini, setelah pernyataan if pertama, kita dapat menambahkan satu atau lebih pernyataan else if untuk memeriksa kondisi tambahan, diakhiri dengan blok else sebagai alternatif jika semua kondisi sebelumnya salah.<br>
**D. Pernyataan Switch Case<br>**
‚Ä¢	Definisi: Pernyataan switch case digunakan untuk memilih satu dari beberapa blok kode untuk dieksekusi berdasarkan nilai dari variabel tertentu. Ini merupakan alternatif yang lebih terstruktur dibandingkan dengan penggunaan banyak if else.<br>
‚Ä¢	Sintaks: Sintaks switch dimulai dengan kata kunci switch, diikuti oleh variabel dalam tanda kurung. Di dalam blok switch, kita mendefinisikan beberapa case yang masing-masing mewakili nilai yang mungkin dari variabel tersebut, diakhiri dengan break untuk menghentikan eksekusi lebih lanjut setelah kasus yang cocok ditemukan.<br>
**E. Operator Ternary<br>**
‚Ä¢	Definisi: Operator ternary adalah cara singkat untuk mengekspresikan pengambilan keputusan dengan cara yang lebih ringkas dibandingkan dengan if else. Ini sering digunakan untuk memberikan nilai berdasarkan kondisi.<br>
‚Ä¢	Sintaks: Sintaks operator ternary terdiri dari kondisi yang diikuti oleh tanda tanya, di mana jika kondisi benar, nilai pertama akan dikembalikan, dan jika salah, nilai kedua akan dikembalikan.<br>
**F. Percabangan Bersarang<br>**
‚Ä¢	Definisi: Percabangan bersarang terjadi ketika satu atau lebih pernyataan if atau else if diletakkan di dalam blok kode dari pernyataan if lainnya. Ini memungkinkan untuk membuat keputusan yang lebih kompleks berdasarkan beberapa kondisi.<br>
‚Ä¢	Sintaks: Sintaks untuk percabangan bersarang mirip dengan pernyataan if biasa, tetapi dengan menempatkan pernyataan if tambahan di dalam blok kode dari if yang sudah ada, memungkinkan evaluasi kondisi yang lebih dalam.<br>

**Penjelasan Soal Latihan :**
1.	Program Kalkulator Sederhana: Program ini meminta pengguna untuk memasukkan dua bilangan dan operator. Berdasarkan operator yang dipilih, program akan melakukan operasi aritmatika yang sesuai dan menampilkan hasilnya.
2.	Program Diskon Pembelian: Program ini menghitung diskon berdasarkan total pembelian yang dimasukkan oleh pengguna, menggunakan beberapa kondisi untuk menentukan persentase diskon yang berlaku.
3.	Program Rumus Luas Permukaan: Program ini menggunakan switch case untuk menghitung luas permukaan berbagai bangun (seperti bola, kubus, dan tabung) berdasarkan pilihan pengguna.
4.	Program Penentuan Nilai Akhir Mahasiswa: Program ini menghitung nilai akhir mahasiswa berdasarkan bobot nilai kehadiran, tugas, UTS, dan UAS, serta menentukan kelulusan berdasarkan rentang nilai yang ditentukan.

# MODUL 6 FLOW CONTROL: LOOP (PERULANGAN)
**Tujuan Pembelajaran :**
1.	Mampu menjelaskan dan mengimplementasikan Looping: Perulangan dalam pemrograman menggunakan IDE.
   
**Penjelasan Singkat Materi:<br>**
**A. Perulangan For<br>**
‚Ä¢	Definisi: Perulangan for adalah jenis perulangan yang digunakan ketika jumlah iterasi sudah diketahui sebelumnya. Ini sering digunakan untuk mengulang blok kode dengan variabel penghitung yang jelas.<br>
‚Ä¢	Sintaks: Sintaks perulangan for terdiri dari tiga bagian dalam tanda kurung: inisialisasi variabel penghitung, kondisi untuk melanjutkan perulangan, dan pernyataan untuk mengubah nilai penghitung setelah setiap iterasi. Blok kode yang ingin diulang diletakkan dalam kurung kurawal.<br>
**B. Perulangan While<br>**
‚Ä¢	Definisi: Perulangan while adalah jenis perulangan yang digunakan ketika jumlah iterasi tidak diketahui dan bergantung pada kondisi tertentu. Perulangan ini akan terus berjalan selama kondisi yang diberikan bernilai benar<br>
‚Ä¢	Sintaks: Sintaks perulangan while dimulai dengan kata kunci while, diikuti oleh kondisi dalam tanda kurung. Blok kode yang akan dieksekusi selama kondisi tersebut benar diletakkan dalam kurung kurawal. Jika kondisi awal sudah salah, blok kode tidak akan dijalankan sama sekali.<br>
**C. Perulangan Do While<br>**
‚Ä¢	Definisi: Perulangan do while mirip dengan perulangan while, tetapi dengan perbedaan bahwa blok kode akan dieksekusi setidaknya satu kali, sebelum kondisi diperiksa. Ini berguna ketika kita ingin memastikan bahwa kode dijalankan setidaknya sekali.<br>
‚Ä¢	Sintaks: Sintaks perulangan do while dimulai dengan kata kunci do, diikuti oleh blok kode yang akan dieksekusi. Setelah blok kode, terdapat kata kunci while diikuti oleh kondisi dalam tanda kurung. Kondisi ini akan diperiksa setelah eksekusi blok kode.<br>
**D. Perulangan Bersarang (Nested Loop)<br>**
‚Ä¢	Definisi: Perulangan bersarang terjadi ketika satu perulangan diletakkan di dalam perulangan lainnya. Ini memungkinkan pembuatan struktur yang lebih kompleks, seperti tabel atau pola.<br>
‚Ä¢	Sintaks: Sintaks untuk perulangan bersarang mirip dengan perulangan biasa, tetapi satu perulangan ditulis di dalam blok kode perulangan lainnya. Ini memungkinkan iterasi ganda, di mana perulangan luar mengatur iterasi yang lebih besar, sementara perulangan dalam mengatur iterasi yang lebih kecil.<br>
**E. Counted Loop dan Uncounted Loop<br>**
‚Ä¢	Definisi: Counted loop adalah perulangan yang jumlah iterasinya sudah ditentukan sebelumnya (seperti perulangan for), sedangkan uncounted loop adalah perulangan yang tidak memiliki jumlah iterasi yang pasti (seperti perulangan while dan do while).<br>
‚Ä¢	Sintaks: Counted loop biasanya menggunakan perulangan for dengan inisialisasi, kondisi, dan increment. Uncounted loop, di sisi lain, menggunakan pernyataan while atau do while dengan kondisi yang menentukan apakah perulangan akan dilanjutkan atau tidak.<br>

<br>

‚Ä¢	Array juga dapat dibuat dalam bentuk multidimensi, seperti array dua dimensi yang sering digunakan untuk membuat matriks. Untuk mengakses elemen dalam array dua dimensi, kita menggunakan dua indeks, contohnya array[i][j].

**I.	Fungsi pada String**

**Penjelasan Singkat Materi:<br>**
**A.	Pengertian Fungsi<br>**
‚Ä¢	Fungsi adalah sub-program yang dapat digunakan kembali dalam program. Fungsi memungkinkan pengorganisasian kode yang lebih baik dan pengurangan pengulangan kode.<br>
**B.	Penggunaan Fungsi<br>**
‚Ä¢	Fungsi sering digunakan untuk melakukan tugas tertentu dan dapat dipanggil berkali-kali dalam program. Contoh fungsi yang umum adalah main(), yang merupakan titik awal eksekusi program.<br>
**C.	Fungsi Rekursif<br>**
‚Ä¢	Fungsi rekursif adalah fungsi yang memanggil dirinya sendiri. Ini berguna untuk menyelesaikan masalah yang dapat dibagi menjadi sub-masalah yang lebih kecil.<br>
**D.	Variabel Lokal & Global<br>**
‚Ä¢	Variabel lokal hanya dapat diakses di dalam fungsi tempat mereka dideklarasikan, sedangkan variabel global dapat diakses dari semua fungsi dalam program. Pemahaman tentang kedua jenis variabel penting untuk menghindari konflik nama dan memanipulasi data dengan benar.<br>

‚Ä¢	Pass by reference mengacu pada pengiriman alamat memori dari variabel ke fungsi. Dengan cara ini, fungsi dapat mengubah nilai variabel asli, karena ia bekerja langsung dengan alamat memori.<br>
**G.	Fungsi dengan Parameter<br>**
‚Ä¢	Fungsi dapat menerima parameter yang memungkinkan pengguna untuk mengirimkan data ke dalam fungsi. Parameter dapat berupa berbagai tipe data, termasuk array dan pointer.<br>
**H.	Fungsi yang Mengembalikan Nilai<br>**
‚Ä¢	Fungsi dapat mengembalikan nilai menggunakan kata kunci return. Tipe data dari nilai yang dikembalikan harus ditentukan saat mendeklarasikan fungsi.<br>

**Penjelasan Singkat Materi:<br>**

‚Ä¢	Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Setiap variabel memiliki alamat memori yang unik, dan pointer digunakan untuk mengakses data di alamat tersebut.<br>
**B.	Penggunaan Pointer<br>**
‚Ä¢	Pointer digunakan untuk mengelola memori secara efisien, memungkinkan akses dan manipulasi data tanpa perlu menyalin nilai, serta untuk mengimplementasikan struktur data dinamis seperti linked list dan tree.<br>
**C.	Deklarasi Pointer<br>**
‚Ä¢	Pointer dideklarasikan dengan menambahkan simbol asterisk (*) sebelum nama pointer. Contoh: int *pointer; menunjukkan bahwa pointer adalah pointer yang menyimpan alamat dari variabel bertipe int.<br>
**D.	Mengambil Alamat Memori<br>**
‚Ä¢	Untuk mendapatkan alamat memori dari variabel, digunakan simbol ampersand (&). Contoh: &variabel akan memberikan alamat memori dari variabel.<br>
**E.	Mengakses Data Menggunakan Pointer<br>**
‚Ä¢	Untuk mengakses nilai yang disimpan di alamat memori yang ditunjuk oleh pointer, gunakan operator dereference (*) di depan nama pointer. Contoh: *pointer akan mengakses nilai yang ada di alamat yang disimpan oleh pointer.<br>
**F.	Pointer dan Array<br>**
‚Ä¢	Pointer sering digunakan untuk mengakses elemen dalam array. Pointer dapat diinkrementasi untuk berpindah ke elemen berikutnya dalam array, memudahkan navigasi dan manipulasi data.<br>
**G.	Pointer untuk Pass by Reference<br>**
‚Ä¢	Pointer memungkinkan penggunaan pass by reference dalam fungsi. Dengan menggunakan pointer sebagai parameter, perubahan yang dilakukan dalam fungsi akan mempengaruhi variabel asli yang dipassing.<br>
**H.	Manfaat Penggunaan Pointer<br>**
‚Ä¢	Penggunaan pointer dapat meningkatkan efisiensi memori, memungkinkan manipulasi data yang lebih kompleks, dan mendukung pengembangan struktur data yang lebih dinamis.<br>
